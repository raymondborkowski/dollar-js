What is it?

DollarJS makes interacting with DOM simple.

It's the only library that you will ever need for finding and manipulating Elements, binding events to your document, dynamically styling your content, and more.

Use it as a jQuery replacement to enjoy most of the same features at a fraction of the size. And unlike competing libraries, DollarJS is modular! Create a custom build for your project with only the modules that you need.

$ npm install dollar-js
